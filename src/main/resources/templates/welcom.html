API

/ - Документация проекта

/user - Получаем весь список пользователей /user

/setuser - Создает нового юзера..Пример запроса: http://localhost:8080/setuser?name=User&userrole=admin&login=login$&pass=RTYUFSGRTER

/getUserID - Поиск юзера по id..Пример запроса: http://localhost:8080/getUserID?id=4

/deleteUserID - Delete юзера по id..Пример запроса: /getUserID?id=4
